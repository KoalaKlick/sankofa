# AfroTix

A modern event management platform built with Next.js, featuring real-time voting, seamless ticket sales, and powerful analytics.

## Getting Started

First, install dependencies:

```bash
pnpm install
```

Then, run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- Real-time event voting and polling
- Seamless ticket sales integration
- Powerful analytics dashboard
- Community event management
- Live webinar support
